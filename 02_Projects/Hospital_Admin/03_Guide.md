# Phase 3. User Guide (사용자 가이드)

## 1. 일일 운영 루틴 (Daily Routine)
1.  **데이터 업로드 (중요)**
    - 매일 오전, 닥터스에서 다운로드한 엑셀 파일(입원/외래)을 구글 드라이브 `닥터스데이터` 폴더에 업로드하세요.
    - **파일명 규칙 필수**: `닥터스_입원_2025-01-05.xlsx` 형식이어야 인식이 됩니다.
2.  **자동 실행 확인**
    - 오후 2시에 로봇이 돌면서 `원무일보` 폴더에 새 파일을 만듭니다.
3.  **결과 확인**
    - 생성된 원무일보 엑셀을 열어 특이사항(수동 입력란)만 채우고 결재를 올리십시오.

## 2. 월초 운영 루틴 (Monthly Routine)
- 매월 1일이 되면 로봇이 자동으로 새로운 `수납여부_YYYY년MM월` 파일을 만듭니다.
- 전월 미수금이 자동으로 이월되었는지 확인하십시오.

## 3. 주의사항 (Caution)
- **발생월 추적 한계**: 금액으로 역추적하므로, 동일 금액의 미수금이 여러 달에 걸쳐 있다면 AI가 가장 최근 달부터 매칭합니다. 특이 케이스는 수동 확인이 필요할 수 있습니다.
- **구글 드라이브 경로**: 폴더 위치나 이름을 마음대로 바꾸시면 에러가 납니다.

## 4. 에러 발생 시 (Troubleshooting)
- **"The file does not exist"**: 오늘 날짜의 닥터스 파일이 업로드되지 않은 것입니다. 파일명을 확인하세요.
- **Aging 계산 오류**: 과거 수납여부 파일 중 누락된 달이 있는지 확인하세요.
