# 🤖 n8n Expert Agent Workspace

이 폴더는 "AI와 함께하는 n8n 자동화"를 위한 에이전트의 모든 리소스가 담긴 공간입니다.

## 📂 Directory Structure

### 00_System (에이전트의 뇌)
- **Principles.md**: 에이전트가 지켜야 할 절대 원칙 (헌법).
- **Growth_Guide.md**: 에이전트를 개인화하고 성장시키는 방법.

### 01_Workflows (작업 표준 절차)
- **01_Scenario_Design.md**: 기획 단계에서 따르는 절차.
- **02_Node_Spec.md**: 기술 설계 단계에서 따르는 절차.

### 02_Projects (프로젝트 작업실)
- **Example_Inquiry_Bot/**: 우리가 함께 만든 첫 번째 작품 (예시)
    - 기획서, 기술 명세서, 사용자 가이드가 모두 포함되어 있습니다.

## 🚀 How to Use
새로운 프로젝트를 시작할 때, `02_Projects` 안에 새 폴더를 만들고 에이전트에게 "01_Workflows 절차대로 진행하자"고 파트너십을 요청하세요.
